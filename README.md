# Street Fighter

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install dependencies.

```bash
npm install
```
## Usage

```bash
npm run build
npm run start
```
Open [http://localhost:8080/](http://localhost:8080/) to start app
